# Dr. Akanksha Dubey Website

A Vite + React + Tailwind website for Dr. Akanksha Dubey.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Create a new GitHub repo, for example `akanksha-dubey-website`.
2. Upload/push these files to the `main` branch.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **GitHub Actions**.
5. Push to `main`; the included workflow will build and deploy the site.

Your URL will usually be:

```text
https://YOUR_GITHUB_USERNAME.github.io/akanksha-dubey-website/
```
